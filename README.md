# AI-INSTAGRAM-ASSISTANT
An AI-powered Instagram engagement assistant that helps creators manage comments and messages using intelligent automated responses.

## Features

- Comment management
- Message management
- AI-generated replies
- Conversation history
- Analytics dashboard

## Tech Stack

- Python
- Flask/FastAPI
- SQLite/MySQL
- Gemini API

## Project Status

Currently in planning and development phase.

- Creator Personality Mode
